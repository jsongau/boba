Report a correction

Wrong hours? A shop that closed or moved? A detail that's off? Tell us and we'll verify it.

Email corrections@niteboba.vercel.app with the shop name, its city, and what's wrong. If you can, include a link to the shop's own hours or page so we can confirm.

We check corrections against the shop's own listing before changing anything, so a report leads to a verified fix rather than a guess.
